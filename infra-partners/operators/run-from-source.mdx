title: "Building a Node from Source"
sidebarTitle: "Run a Node from Source"
---

Docker images are the easiest way to run a Celo node, but you can always build your own node from source code. You might wish to do this if you want to run on a specific architecture or inspect the source code.

The [celo-l2-node-docker-compose](https://github.com/celo-org/celo-l2-node-docker-compose) codebase is still the best reference for how to run your nodes from source, and the [Network Config & Assets](/infra-partners/operators/network-config) page lists everything you need to participate in the network.
