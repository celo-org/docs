## Enabling Testnet in MiniPay

---

1. This is how the homescreen looks like, you can swipe down to get to the next screen.

<img
  src={"/img/doc-images/minipay/enabling-testnet/minipay-1.png"}
  style={{ width: 250 }}
  alt="Open MiniPay dApp store"
/>

2. Tap the settings icon to open up the settings menu.

<img
  src={"/img/doc-images/minipay/enabling-testnet/minipay-2.png"}
  style={{ width: 250 }}
  alt="Open MiniPay dApp store"
/>

3. Tap on the "About" option to get to the about screen.

<img
  src={"/img/doc-images/minipay/enabling-testnet/minipay-3.png"}
  style={{ width: 250 }}
  alt="Open MiniPay dApp store"
/>

4. Tap on the "Version" number until you see the message that you have enabled developer mode.

<img
  src={"/img/doc-images/minipay/enabling-testnet/minipay-4.png"}
  style={{ width: 250 }}
  alt="Open MiniPay dApp store"
/>

5. If you go back to the settings menu now, you will see an additional option of "Developer Settings" tap on it.

<img
  src={"/img/doc-images/minipay/enabling-testnet/minipay-5.png"}
  style={{ width: 250 }}
  alt="Open MiniPay dApp store"
/>

6. Now you will see the option to enable testnet!

<img
  src={"/img/doc-images/minipay/enabling-testnet/minipay-6.png"}
  style={{ width: 250 }}
  alt="Open MiniPay dApp store"
/>
