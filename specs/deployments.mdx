## Mainnet

Celo Mainnet will be migrated to an L2 on March 26, 2025, around 3:00 AM UTC. The migration is scheduled for block *31056500*. For this L1 chain will be stopped at a block height of *31056499*. At this point the existing state will migrated to work with the L2 nodes. The migration process preserves the full L1 history while updating it to work with the Celo L2 stack. More technical details are available in the [migration docs](/specs/l2-migration).

The Celo L2 network has the following chain properties:

- Block period: 1 second
- Block gas limit: 30,000,000 gas
- Block gas target: 6,000,000 gas (see EIP-1559 elasticity multiplier)
- EIP-1559 elasticity multiplier: 5
- EIP-1559 denominator: 400
- EIP-1559 floor: 25 Gwei
